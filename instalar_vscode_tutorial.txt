commit 1: Iniciando tutorial do VSCode

 Bom dia!, neste tutorial irei explicar sobre o VSCode e mostrar passo a passo da instalação do VSCode no computador ou maquina virtual.


commit 2: Explicando o que é VSCode

 O VSCode, ou Visual Studio Code, é um editor de código-fonte desenvolvido pela Microsoft. É gratuito, de código aberto e disponível para Windows, macOS e Linux. Ele suporta uma variedade de linguagens de programação e frameworks e é altamente personalizável, permitindo aos usuários instalar extensões para adicionar novas funcionalidades. É conhecido por sua interface limpa e eficiente e por recursos como depuração integrada, controle de versão Git e um terminal integrado.


commit 3: Passo a Passo para o download

1. Abra um navegador da web.

2. Acesse o site oficial do Visual Studio Code em https://code.visualstudio.com/

3. Clique no botão "Download" para baixar o instalador adequado para o seu sistema operacional (Windows, macOS ou Linux).


commit 4: Passo a Passo de instalação

1. Após o download ser concluído, localize o arquivo de instalação (normalmente na pasta de downloads) e clique duas vezes para iniciá-lo.

2. Siga as instruções do instalador para configurar o Visual Studio Code em seu computador. Você pode manter as configurações padrão ou personalizá-las de acordo com suas preferências.

3. Durante a instalação, você pode optar por adicionar o VSCode ao PATH do sistema para acessá-lo facilmente a partir da linha de comando.

4. Clique em "Next" para avançar através das etapas do instalador.

5. Quando a instalação estiver concluída, clique em "Finish" para sair do instalador.

6. Depois de instalado, o VSCode pode ser encontrado no menu Iniciar ou na área de trabalho. Clique duas vezes no ícone para abrir o VSCode.


commit 5: Configuração inicial VSCode 

1. Quando você abrir o VSCode pela primeira vez, pode ser solicitado que você instale algumas extensões recomendadas. Você pode optar por instalar ou ignorar essas extensões por enquanto

2. Você também pode personalizar suas preferências, como tema, fonte, atalhos de teclado, entre outros, acessando as configurações do VSCode.
